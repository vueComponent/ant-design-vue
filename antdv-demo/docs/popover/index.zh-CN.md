## API

| 参数    | 说明     | 类型                | 默认值 |
| ------- | -------- | ------------------- | ------ |
| content | 卡片内容 | string\|slot\|VNode | 无     |
| title   | 卡片标题 | string\|slot\|VNode | 无     |

更多属性请参考 [Tooltip](/components/tooltip-cn/#API)。

## 注意

请确保 `Popover` 的子元素能接受 `mouseenter`、`mouseleave`、`focus`、`click` 事件。
