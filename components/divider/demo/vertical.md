<cn>
#### 垂直分割线
使用 `type="vertical"` 设置为行内的垂直分割线。
</cn>

<us>
#### Vertical
Use `type="vertical"` make it vertical.
</us>

```tpl
<template>
  <div>
    Text
    <a-divider type="vertical" />
    <a href="#">Link</a>
    <a-divider type="vertical" />
    <a href="#">Link</a>
  </div>
</template>
```
